Periodic Table Research Project

A school project by Ethan Manzi

This is my online version of my project. I choose this, because working with all this is far easier in HTML, in my opinion. Instead of having to download anything I need, create a Pages document, export it either as a Word document (and risk it not converting properly), or export it as a PDF. I figure this way, in this format, I can actually use this as a tool for the class!
There's a few pages you can go to here. If you want all this data formated like a normal printable document, click this link. If you want to go right to the "atomic radius / atomic number" graph, then use this link, and to go right to the "ionization energy / atomic number" graph, then use this link.

In the sources, you may notice Wikipedia is listed. For some reason, may teachers despise this source. For the context of this assessment, no data from it is used. As stated, I added extra information to this, to make it a resource that I can use after this assessment. Thusly, I needed to pull extra data from somewhere, and love it or hate it, Wikipedia has vast wealths of data.
